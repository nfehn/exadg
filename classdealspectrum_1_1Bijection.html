Detailed Description

Class for mapping between space filling curve (at the moment only morton curve) and lexicographical order. It furthermore contains the partitioning information of the sfc among processes.

◆ init() [1/2]

void dealspectrum::Bijection::init ( )
inline

Initialize mapping data structures in the case that no process local cell count is given. We distribute cells over all processes evenly: #cells / #processes. The remainder cells (#cells % #processes) are assigned to the first processes.
